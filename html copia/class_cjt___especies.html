◆ llegir_conjunt_especies()

void Cjt_Especies::llegir_conjunt_especies ( int  n)

Llegim des del canal d'entrada un conjunt d'espècies.

Precondición
Llegim al canal d'entrada un enter n >= 0 i seguidament un conjunt d'espècies, les n espècies, tenen dos strings associats, un identificador i un gen, cadascún únic per a cada espècies diferent. És a dir dues espècies, seran diferents si, i només si, tenen un gen o un identificador diferents. Els continguts previs del conjunt d'espècies els esborrem.
Postcondición
El resultat serà un conjunt n d'espècies que passaran a formar part del conjunt d'espècies.
